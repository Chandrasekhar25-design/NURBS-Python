Control Points Manager
^^^^^^^^^^^^^^^^^^^^^^

The ``control_points`` module provides helper functions for managing control points. It is a better alternative to
the :doc:`compatibility module <module_compatibility>` for managing control points. Please refer to the examples
below for more details.
