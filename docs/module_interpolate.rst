Interpolation
^^^^^^^^^^^^^

``interpolate`` module provides functions for interpolating B-spline curves and surfaces from points. Currently, this
module only applies global interpolation to a list of data points. Please see the following functions for details:

* :py:func:`.interpolate_curve`
* :py:func:`.interpolate_surface`
